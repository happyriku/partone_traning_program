# 個人会員登録API

## 概要

```txt
個人会員登録APIは，ユーザーが会員情報を入力して，会員登録処理を実行するものである。
```

<br>

## 機能

1. [1-会員情報をチェックするもの](#1-会員情報をチェックするもの)

<br>


## 詳細

### 1-会員情報をチェックするもの
**メソッド : POST**

<br>

**引数**

|番号|名称|型|説明|
|:--|:--|:--|:--|
|01|会員情報|-|ユーザーが入力した会員情報|

**画面遷移**

- 会員情報の登録に成功した場合

> メールアドレスを入力する画面に遷移する

- 会員情報の登録に失敗した場合

> 入力していない項目や登録条件に合致していない項目にエラー文を出力する
